De database
U kunt in deze database zoeken op 4 verschillende dingen:
- Op de header, zodat als u zelf wat toevoegt, de resultaten gemakkelijk terug kan vinden
- Op het naam van een organisme
- Op de eiwit functie
- Op de accessie code
Als u de muis houd op sequentie ID, kunt u de header zien welke bij dat resultaat hoort
Als u op de accessiecode klikt wordt u doorgelinkt naar de website van NCBI die bij deze accessiecode hoort.

Op deze website kan je in de database filteren.
Voer een categorie in en een zoekwoord.
Klik daarna op zoek om in de database te zoeken.    (voor meer informatie, klik op het vraagteken rechtsboven)
